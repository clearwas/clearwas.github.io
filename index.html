Vim vixen 插件快捷键

Basic usage

  • j, k: 向下/上滚动
  • h, l: scroll horizontally
  • Ctrl+U, Ctrl+D: 向上/向下滚动半屏
  • Ctrl+B, Ctrl+F: 向上/向下滚动一屏
  • 0, $: 滚动到最左端/右端
  • gg, G: 滚动到顶部/底部
  • d: 删除当前标签页
  • u:重开删除的标签页 reopen close tab
  • K, J: 向前/后切换标签页
  • r: 重新加载页面
  • zp: 切换固定/取消固定当前标签
  • zd: 复制当前标签

-f：开始关注页面中的链接
-H：回顾历史
-L：回顾历史
-[[，]]：找到上一个或下一个链接并打开它
-gu：转到父目录
-gU：转到根目录

Anaconda介绍

发表于 2020-04-16 分类于 挖坑

Anaconda

功能：

包管理，在conda基础上开发而来
环境管理，功能类似 Virtualenv，解决了多版本Python并存、切换的问题

conda工具介绍

conda是Anaconda下用于包管理和环境管理的工具,功能上类似pip和vitualenv的组合。安装成功后conda会默认加入到环境变量中，因此可直接在命令行窗口运行命令conda
conda的环境管理与virtualenv是基本上是类似的操作。

相关操作

```
#查看帮助
conda -h
#基于python3.6版本创建一个名字为python36的环境
conda create –name python36 python=3.6
#激活环境
activa python36
# 再来检查python版本，显示是 3.6
python -V
# 退出当前环境
deactive
