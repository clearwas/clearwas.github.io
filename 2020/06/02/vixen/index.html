Vim vixen 插件快捷键

Basic usage

  • j, k: 向下/上滚动
  • h, l: scroll horizontally
  • Ctrl+U, Ctrl+D: 向上/向下滚动半屏
  • Ctrl+B, Ctrl+F: 向上/向下滚动一屏
  • 0, $: 滚动到最左端/右端
  • gg, G: 滚动到顶部/底部
  • d: 删除当前标签页
  • u:重开删除的标签页 reopen close tab
  • K, J: 向前/后切换标签页
  • r: 重新加载页面
  • zp: 切换固定/取消固定当前标签
  • zd: 复制当前标签

-f：开始关注页面中的链接
-H：回顾历史
-L：回顾历史
-[[，]]：找到上一个或下一个链接并打开它
-gu：转到父目录
-gU：转到根目录
